RTS Benchmark

One prompt, one file, every new model.

Each new model gets the same prompt — verbatim, one attempt — to build a real-time strategy game in a single HTML file. No edits, no follow-ups. What it produces is what you play.

“Create a simple but functional real time strategy game similar to old WarCraft, StarCraft or Command & Conquer. Build buildings, create units, gather resources, uncover the map. No AI, no multiplayer, no sound. Everything in a single HTML/CSS/JS file.”
